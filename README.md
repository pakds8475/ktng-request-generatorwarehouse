# Генератор заявок KT&G

Одностраничный сайт в стиле простого Vercel-инструмента: пользователь заполняет форму, добавляет строки груза и скачивает готовую заявку в формате DOCX.

## Что умеет

- Создает заявку на выдачу товара.
- Создает заявку на приемку.
- Подставляет город, дату, время, номер автомобиля/контейнера, ФИО водителя.
- Добавляет несколько строк груза: наименование, вес, количество.
- Сохраняет подписи сторон и стандартные реквизиты.
- Имеет кнопки с примерами по вашим файлам: Андижан на выдачу и Самарканд приемка.

## Запуск локально

```bash
npm install
npm run dev
```

После запуска откройте адрес, который покажет Vite, обычно `http://localhost:5173`.

## Деплой на Vercel

1. Загрузите эту папку в GitHub.
2. В Vercel выберите **Add New Project**.
3. Выберите репозиторий.
4. Vercel автоматически увидит Vite-проект.
5. Нажмите **Deploy**.

## Где менять шаблон DOCX

Основная логика генерации находится в файле:

```text
src/docxGenerator.js
```

Интерфейс и поля формы находятся в:

```text
src/App.jsx
```
